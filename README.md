# 🏋️ FitClub - Fitness & Gym Website

A responsive fitness and gym website built with **React**, featuring smooth animations and a modern energetic UI.

## 🔗 Live Demo
[View Live →](https://fitclubwebsitebyhariscoding.pages.dev/)

## 📸 Screenshot
![FitClub Screenshot](screenshot.png)

## 🛠️ Built With
- React
- CSS3 (Animations)
- Vite
- Deployed on Cloudflare Pages

## 🚀 Getting Started

### Installation
```bash
git clone https://github.com/HarisShahnawaz/FitClub-reactApp
cd FitClub-reactApp
npm install
npm run dev
```

## ✨ Features
- Responsive design for all screen sizes
- Smooth scroll animations
- Modern gym-themed UI
- Mobile-friendly navigation
